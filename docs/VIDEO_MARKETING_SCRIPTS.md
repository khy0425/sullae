# 술래 앱 - 숏폼 영상 마케팅 스크립트

## 개요
- 타겟 플랫폼: TikTok, Instagram Reels, YouTube Shorts
- 영상 길이: 15-30초
- 목표: 앱 다운로드 유도, 바이럴 확산

---

## 템플릿 A: 문제-해결형 (Problem-Solution)

### A-1. "주말에 뭐하지?" 버전

**[0-3초] 훅 (Hook)**
- 화면: 소파에 누워 폰만 보는 모습 / 지루한 표정
- 자막: "주말인데 또 집에만..."
- 음악: 느린 Lo-fi 비트

**[3-8초] 문제 강조**
- 화면: 카톡 그룹채팅 "주말에 뭐해?" "몰라" "집" 대화
- 자막: "친구들도 다 집순이 집돌이"
- 음악: 계속 느린 비트

**[8-12초] 전환점**
- 화면: 술래 앱 알림 팝업 "근처에 술래잡기 모임이 있어요!"
- 자막: "근데 이거 뭐지?"
- 음악: 비트 드롭 시작

**[12-22초] 해결 (앱 시연)**
- 화면: 앱 실행 → 모임 목록 → 참가 → 실제 게임 장면
- 자막:
  - "동네에서 술래잡기 모임 찾고"
  - "참가 버튼 하나로 끝"
  - "어른이 되어서 술래잡기 ㄹㅇ 꿀잼"
- 음악: 신나는 비트

**[22-28초] CTA**
- 화면: 앱 다운로드 화면 + 플레이 중인 사람들
- 자막: "술래 - 다운받고 밖에서 놀자 🏃"
- 음악: 페이드아웃

---

### A-2. "운동하기 싫어" 버전

**[0-3초] 훅**
- 화면: 헬스장 앞에서 망설이는 모습
- 자막: "운동해야 하는데... 귀찮아"

**[3-8초] 문제**
- 화면: 런닝머신 위 지루한 표정
- 자막: "헬스장 1시간 = 지옥"

**[8-12초] 전환**
- 화면: 공원에서 뛰어다니는 사람들
- 자막: "근데 얘네는 왜 저렇게 신나?"

**[12-22초] 해결**
- 화면: 술래 앱 → 참가 → 실제 술래잡기 장면
- 자막:
  - "술래잡기 30분에 3km 뜀 ㅋㅋ"
  - "근데 진짜 재밌어서 힘든 줄도 모름"

**[22-28초] CTA**
- 화면: 앱 로고 + "이번 주말 같이 뛰자"
- 자막: "술래 앱 다운로드 👇"

---

## 템플릿 B: 스토리텔링형 (Mini Story)

### B-1. "어린 시절 추억" 버전

**[0-5초] 과거 회상**
- 화면: 어린이 놀이터 / 해질녘 놀던 추억 (AI 이미지 or 실제)
- 자막: "어릴 때 해질 때까지 술래잡기 했던 거 기억나?"
- 음악: 감성적인 피아노

**[5-10초] 현실**
- 화면: 어른이 된 지금, 폰만 보는 일상
- 자막: "어른 되니까 그런 거 없더라"

**[10-15초] 발견**
- 화면: 술래 앱 발견
- 자막: "근데 이런 앱이 있었음"

**[15-25초] 체험**
- 화면: 앱으로 모임 참가 → 실제 게임 장면 (웃고 뛰는 모습)
- 자막:
  - "동네 사람들이랑 진짜 술래잡기 함"
  - "ㅋㅋㅋ 어른인데 이렇게 뛰어본 거 오랜만"

**[25-30초] CTA**
- 화면: 앱 로고 + 다운로드 유도
- 자막: "술래 - 어른들의 놀이터"

---

### B-2. "새로운 친구" 버전

**[0-3초] 훅**
- 화면: 혼자 공원 벤치에 앉아있는 모습
- 자막: "이사 오고 아는 사람 0명..."

**[3-8초] 문제**
- 화면: 빈 카톡 채팅방
- 자막: "동네 친구 어떻게 사귀지?"

**[8-15초] 발견과 참가**
- 화면: 술래 앱 → 근처 모임 → 참가
- 자막: "동네 술래잡기 모임이 있다길래 참가해봄"

**[15-25초] 결과**
- 화면: 여러 사람들과 함께 뛰고 웃는 장면
- 자막:
  - "처음엔 어색했는데"
  - "같이 뛰다 보니까 금방 친해짐"
  - "지금은 매주 만나는 사이 ㅋㅋ"

**[25-30초] CTA**
- 화면: 단체 셀카 → 앱 로고
- 자막: "술래에서 동네 친구 만들기"

## 자막 톤 가이드

### 1. MZ 캐주얼 톤 (메인)
- "ㅋㅋㅋ", "ㄹㅇ", "꿀잼", "개웃김"
- 반말 사용
- 예: "이거 진짜 미침 ㅋㅋㅋ"

### 2. 감성 톤 (스토리텔링용)
- 물음표로 공감 유도
- 짧은 문장
- 예: "어릴 때 그거 기억나?"

### 3. 정보 전달 톤 (기능 소개용)
- 간결하고 명확
- 숫자 활용
- 예: "30분 만에 3km 달림"

## 앱 화면 녹화 체크리스트

### 필수 촬영 장면
- [ ] 앱 스플래시 화면 (로고 애니메이션)
- [ ] 홈 화면 (모임 목록)
- [ ] 지도에서 근처 모임 표시
- [ ] 모임 상세 화면
- [ ] 참가하기 버튼 클릭
- [ ] 참가 완료 애니메이션
- [ ] 채팅 화면
- [ ] 게임 진행 화면 (술래 선정, 타이머 등)

### 촬영 설정
- 해상도: 1080x1920 (9:16 세로)
- 프레임: 60fps 권장
- 다크모드/라이트모드 둘 다 촬영
- 알림바 숨기기 (화면 녹화 전)

### 데모 데이터 준비
- 가상 사용자 프로필 3-5개
- 모임 데이터 2-3개 (오늘/내일 일정)
- 채팅 메시지 미리 작성
- 참가자 수 적당히 (4-8명)

---

## 실제 게임 장면 촬영 팁

### 장소
- 넓은 공원 (잔디밭)
- 해질녘 골든아워 추천
- 배경에 사람 많으면 더 좋음

### 촬영 구도
- 달리는 장면 (슬로모션)
- 숨는 장면 (POV)
- 잡히는 순간 (리액션)
- 단체 하이파이브
- 다 같이 웃는 장면

### 참가자
- 20-30대 4-6명
- 밝은 색 옷 권장
- 자연스러운 리액션 중요
