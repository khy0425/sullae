# 술래 제품 전략

## 핵심 포지셔닝

> **술래는 '게임 앱'이 아니라 '게임 진행 도구'다.**

### 사용 시점
| 상황 | 앱 사용 | 설명 |
|------|---------|------|
| 모임 전 | O | 생성, 참가, 역할 선택 |
| 게임 시작 | O | 팀 배정 확인, 타이머 시작 |
| 게임 중 | **X** | 뛰어다니느라 폰 볼 시간 없음 |
| 라운드 종료 | O | 결과 확인, 다음 라운드 |
| 게임 종료 | O | 최종 결과, 다음 모임 예고 |

**→ 게임 중 화면 상시 사용 ❌ / 시작·종료·공지 시만 사용 ⭕**

이 포지션을 유지하면 UX가 깨지지 않는다.

## 커스텀 룰 프리셋 시스템

### 문제
커스텀 게임은 좋지만, 한 번 쓰고 끝나면 약하다.

### 해결: 룰 프리셋
```
┌─────────────────────────────────────┐
│  내 프리셋                           │
├─────────────────────────────────────┤
│  🏃 우리 동네 경찰과 도둑 룰          │
│     - 감옥: 농구장 벤치              │
│     - 시간: 15분                     │
│     - 도둑 부활: 동료 터치           │
├─────────────────────────────────────┤
│  🏃 러닝용 깃발뺏기                   │
│     - 영역: 공원 전체               │
│     - 깃발 위치: GPS 기반            │
├─────────────────────────────────────┤
│  ❄️ 초보자용 얼음땡                   │
│     - 술래: 2명                      │
│     - 해동: 5초 터치                 │
└─────────────────────────────────────┘
        ↓
   [원클릭 재사용]
```

### 프리셋 데이터 모델
```dart
class GamePreset {
  final String id;
  final String name;
  final String? description;
  final String creatorId;
  final GameType baseGameType;
  final Map<String, dynamic> rules;  // 커스텀 룰
  final bool isPublic;               // 공개 여부
  final int usageCount;              // 사용 횟수
  final DateTime createdAt;
}
```

### 룰 옵션 예시
```dart
// 경찰과 도둑
{
  'roundTime': 15,           // 라운드 시간 (분)
  'jailLocation': '농구장',   // 감옥 위치
  'rescueMethod': 'touch',   // 구출 방법 (touch/highfive/tag)
  'copRatio': 0.3,           // 경찰 비율
}

// 얼음땡
{
  'seekerCount': 2,          // 술래 수
  'unfreezeTime': 5,         // 해동 시간 (초)
  'unfreezeMethod': 'hold',  // 해동 방법 (touch/hold/crawl)
}

// 깃발뺏기
{
  'flagCount': 1,            // 깃발 수
  'useGps': true,            // GPS 기반 영역
  'captureMethod': 'carry',  // 운반 방식
}
```

## UGC (User Generated Content) 진화 로드맵

### Phase 1: 개인 프리셋
- 내가 만든 룰 저장
- 다음 모임에서 재사용

### Phase 2: 공유 프리셋
- 프리셋 공개 설정
- 다른 유저가 복사해서 사용
- "인기 프리셋" 랭킹

### Phase 3: 커뮤니티
- 동네별 인기 룰
- 프리셋 리뷰/평점
- 크리에이터 팔로우

### Phase 4: 공식 게임화
- 인기 UGC → 공식 게임 타입으로 승격
- 크리에이터 인센티브

## 경쟁 우위

### vs 당근 모임
| 요소 | 당근 | 술래 |
|------|------|------|
| 게임 룰 | 없음 | 프리셋으로 명확화 |
| 재사용성 | 매번 새글 | 원클릭 재생성 |
| 팀 구성 | 현장에서 협의 | 사전 희망 + 자동 배정 |
| 게임 진행 | 없음 | 타이머, 라운드 관리 |

### 핵심 차별점
1. **프리셋 = 게임의 표준화**
   - "우리 룰" → 저장 → 공유 → 커뮤니티

2. **진행 도구 = 최소 개입**
   - 게임 중 폰 안 봄
   - 시작/종료만 터치

3. **희망 기반 팀 배정**
   - 사전에 역할 선택
   - 현장 협의 시간 절약

## 핵심 메트릭

### North Star Metric
**월간 완료 게임 수** (모집 → 시작 → 종료까지 완료된 게임)

### 보조 지표
- 프리셋 재사용률
- 평균 참가자 수
- 호스트 → 재호스팅 전환율
- 참가자 → 호스트 전환율
